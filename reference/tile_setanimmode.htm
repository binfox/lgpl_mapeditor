Blitzbasic mapengine function
tile_setanimmode (tile.tile, animation, mode)

Description
This function set or change animation mode/direction. Possible values are: 1=paused, 2=forward, 3=backward.

Parameter
tile.tile=tileset type handle
animation=animation number (1-65535 max)
mode=animation mode (1=paused, 2=forward, 3=backward)

Return value
none

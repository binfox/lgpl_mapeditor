Blitzbasic mapengine function
tile_setframeval (tile.tile, frame, value)

Description
This function set default value of a tileset-frame. In mapeditor, you can assign a short value 0-255 to each animation or frame. With this function, you can modify this value.

Parameter
tile.tile=tileset type handle
frame=frame number (1-65535 max)
value=default value 0-255

Return value
none

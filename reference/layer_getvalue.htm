Blitzbasic mapengine function
value=layer_getvalue (layer.layer, x, y)

Description
Return tile-value from tile coordinate X,Y. You need layer type handle (search with layer_find).

The returned value is not a original "hardcodet" value. This value is "splitted" to frame- or animation- number (1st animation is number 1 and 1st frame is also number 1).

Parameter
layer.layer=layer type handle
x=tile coordinate x (0...)
y=tile coordinate y (0...)

Return value
tile value (0-65535)

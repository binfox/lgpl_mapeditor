Blitzbasic mapengine function
layer_setvalue (layer.layer, x, y, value, code)

Description
Set tile-value at tile coordinate X,Y. You need layer type handle (search with layer_find).

The value is not a original "hardcodet" value. This value is "splitted" to frame- or animation- number (1st animation is number 1 and 1st frame is also number 1).

Parameter
layer.layer=layer type handle
x=tile coordinate x (0...)
y=tile coordinate y (0...)
value=tile value (0-15 or 0-255 or 0-4095 or 0-65535)
code=0=void, 1=frame, 2=anim

Return value
none

Blitzbasic mapengine function
coll=layer_map_collision (layer.layer, spritex, spritey, sprite [,spriteframe])

Description
This function check pixel perfect collision between image and map-layer. You need layer type handle (search with layer_find).

To check a collision between image-layer and map-layer, please use layer_image_collision().

Parameter
layer.layer=layer type handle
spritex=screen coordinate x
spritey=screen coordinate y
sprite=image handle
spriteframe=image frame (default=0)

Return value
0=no collision
1=collision

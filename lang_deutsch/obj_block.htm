Objekt (Block)
Ein Blockobjekt ist ein 1x1 Layer - es ist jedoch viel schneller. Anders als Bildobjekte können diese Objekte nicht beliebig positioniert werden. Ein Blockobjekt kann nur in einem Layer-Raster verschoben werden. Blockobjekte können zudem unabhängig von der Meister-Animation abgespielt werden (anderes Timing).



Layer:
Ein Layer dient als Raster für die Verschiebung.

Tileset:
Ein Tileset kann mit Tileset Editor angelegt und hier jederzeit ausgewählt werden.

Animation:
Hier kann Animationsrichtung eingestellt werden (Pause, Vorwärts, Rückwärts, Standard) - jedoch nur, wenn eine Animation zugewiesen wurde!

Startframe:
Hier kann Startframe eingestellt werden - jedoch nur, wenn eine Animation zugewiesen wurde!

Bildname:
Bildname ist immer optional. In einem Spiel kann man so einzelne Objekte unterscheiden.

Tilepos X/Y:
Definiere hier die Position auf dem Referenz-Layer.

Verschieb X/Y:
Ein Blockobjekt kann um paar Pixel verschoben werden - das nennt sich Justierung.

Grafik maskieren:
Tileset-Grafik kann maskiert dargestellt werden. Normalerweise ist diese Option aktiviert.

Diverses
Hier eine Erklärung der restlichen Symbole:

Neu
Eine Karte wird komplett verworfen - alle Objekte, alle Tilesets und alle Daten werden gelöscht. Nur Zwischenablage bleibt erhalten. Rückgängig ist nicht mehr möglich.

Laden
Blendet einen Dateidialog, über den eine MAP-Datei geöffnet werden kann. Wurde eine Karte verschlüsselt, dann wird zusätzlich ein Dialog zur Passworteingabe eingeblendet.

Speichern
Sichert komplette Karte auf einem Datenträger. Halte linke Shift-Taste fest, um geöffnete Datei ohne Dateiauswahldialog sofort zu speichern.

Einstellungen
Blendet ein Menü zur Auswahl von Farben und Interface-Sprache.



Hilfe
Blendet diese Onlinehilfe ein.

Objekt (Bild)
Ein Bildobjekt (oder auch Sprite genannt) ist ein 1x1 Layer - es ist jedoch viel schneller. Diese Objekte können pixelgenau positioniert werden. Bildobjekte können zudem unabhängig von der Meister-Animation abgespielt werden (anderes Timing).



Tileset:
Ein Tileset kann mit Tileset Editor angelegt und hier jederzeit ausgewählt werden.

Animation:
Hier kann Animationsrichtung eingestellt werden (Pause, Vorwärts, Rückwärts, Standard) - jedoch nur, wenn eine Animation zugewiesen wurde!

Startframe:
Hier kann Startframe eingestellt werden - jedoch nur, wenn eine Animation zugewiesen wurde!

Bildname:
Bildname ist immer optional. In einem Spiel kann man so einzelne Objekte unterscheiden.

Bildpos X/Y:
Bildposition wird in Pixel angegeben

Parallax X/Y:
Parallax ist normalerweise immer 100%. 1%-99% lassen es langsamer scrollen. 101%-200% lassen es schneller scrollen. 0% fixiert den Layer.

Grafik maskieren:
Tileset-Grafik kann maskiert dargestellt werden. Normalerweise ist diese Option aktiviert.

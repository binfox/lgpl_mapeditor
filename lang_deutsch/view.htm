Ansicht
Für bessere Übersicht können einige Details im Map Editor ausgeblendet werden. Folgende Symbole stehen zur Verfügung:

Einzelansicht ein/aus
Wenn eine komplexe Karte mit mehreren Layerschichten bearbeitet wird, dann können viele Tiles nicht mehr vernünftig gesehen werden. Diese Funktion blendet komplett alle Objekte aus und zeigt nur einen Layer an.

Zoom ein/aus
Anzeige erscheint in 3-facher Vergrößerung. So sind selbst feine Details gut erkennbar.

Gitter ein/aus
Layergitter werden nur bei aktiven Layer-Objekten eingeblendet. Wenn ein Layer bearbeitet wird, kann es etwas störend sein. Damit werden alle Gitter ausgeblendet. Objektumrandung bleibt jedoch immer sichtbar.

Parallax ein/aus
Eine Karte mit vielen Objekten und gesetzten Parallax-Werten kann manchmal nicht so einfach bearbeitet werden. So muss man manchmal sehr weit scrollen. Parallax-Scrolling kann jedoch zur besseren Übersicht ausgeschaltet werden.

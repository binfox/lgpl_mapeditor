Copy&Paste
Der Map-Editor enthält ausgeklügeltes Copy&Paste-System. Wer mit MS Excell gearbeitet hat, der wird sich recht schnell daran gewöhnen. Hier eine kurze Erklärung der Symbole:

Markierung ausschnieden

Markierung kopieren / Objekt duplizieren

Markierung einfügen

Hier aber einige Tipps, die man nicht sofort ersehen kann. Erstelle ganz normal deinen Layer und setze einige Tiles rein. Mit dem Markieren-Werkzeug kann ein Bereich selektiert werden. Klicke dann auf Kopieren. Die ausgewählten Tiles werden in einen separaten Speicher kopiert. Selektiere dann ein Tile im Layer und gebe somit die Start-Einfügeposition vor. Klicke dann auf Einfügen. Der Map-Editor kann jedoch mehr! Wenn eine Fläche ausgewählt wird, dann werden Tiles aus der Zwischenablage der Selektion angepasst. Eingefügter Ausschnitt sieht dann gestreckt aus. Für gute Ergebnisse muss mindestens ein 3x3 Feld kopiert werden.



Außerdem verfügt der Map-Editor über einen "Shift compensation"-Algorithmus. Diese Funktion verhindert das "Zerfallen" von Tiles beim Einfügen auf Isometrisch2-, Hexagonal1- und Hexagonal2-Layern. Es darf jedoch nur eine Start-Einfügeposition vorgegeben werden. Um diese Funktion zu umgehen, muss ein Bereich selektiert werden.



Weiterhin erlaubt der Map-Editor das Duplizieren von kompletten Layern und Objekten. Dazu darf keine Markierung auf ein Layer angewendet werden. Klicke dann einfach auf Kopieren und es erscheint ein exaktes Duplikat im Objekt-Menü.



Es ist zudem möglich Ausschnitte aus einem Layer auf einen anderen Layer zu übertragen. Inhalt der Zwischenablage geht nach Öffnen einer anderen Datei NICHT verloren. Jedoch wird eine Warnung eingeblendet. Beachte: Tileset muss gleich oder ähnlich sein - Anzahl der Animationen muss gleich sein.

Halte linke STRG-Taste beim Einfügen, um das Einfügen von leeren Tiles zu unterdrücken.

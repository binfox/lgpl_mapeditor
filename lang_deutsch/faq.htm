Fragen
  • Kann ich die erstellten Karten in kommerziellen Spielen benutzen?
    Ja, sowohl mit FREEversion als auch mit PROversion erstellte Karten.

  • Kann ich die FREEversion des Map-Editors meinem Spiel beilegen?
    Die FREEversion kann jedem Spiel beigelegt werden. Die PROversion kann aus Lizenz- und Copyright-Gründen nicht einfach beigelegt werden.

  • Muss ich in meinen Programmen auf diesen Editor hinweisen?
    Wenn FREEversion benutzt wurde, dann sollte ein Link (www.mapeditor.de.vu) im Spiel, Dokumentation oder Website erscheint. Bei PROversion kann es entfallen.

  • Ich habe die Vollversion gekauft. Darf ich eine Sicherheitskopie auf CD brennen?
    Das ist erlaubt und MUSS sogar gemacht werden. Ein Verlust der Vollversion kann nicht ersetzt werden.

  • Ich habe mit einem anderen Map-Editor eine Map-Datei erstellt. Kann ich diese Datei laden?
    Das geht natürlich nicht. Dieser Editor akzeptiert nur eigenes Dateiformat.

  • Ich habe mehrere Karten mit der FREEversion erstellt. Kann ich diese später mit PROversion laden?
    Das geht problemlos. FREEversion kann jedoch keine PRO-Karten laden!

  • Ich weiß nicht genau wie ich den Code schreiben muss. Bitte mach es für mich.
    Leider gibt es keinen kostenlosen Programmiersupport. Dies würde das Programm deutlich teuerer machen. Zudem kann die Map-Datei in unzähligen Sprachen ausgelesen werden. Viele Probleme sind auch sehr individuell.

  • Kann ich Karten ähnlich "SimCity 2000" oder "Marble Madness" mit diesem Editor erstellen?
    Das ist nur begrenzt machbar. Dieser Editor wurde für flache Karten geschrieben. Es ist zwar auch möglich Tiefe in die Karten zu bringen, für solche Karten ist es jedoch nicht ganz geeignet. Ein Tile in "Marble Madness" muss einzeln gesetzt werden - es werden darum X,Y,Z-Koordinaten, Tilenummer und Z-Order-Werte benötigt. Alle Tiles müssen zudem vorsortiert werden. Wie man sieht werden komplett andere Algorithmen und spezielle Editor-Features benötigt.

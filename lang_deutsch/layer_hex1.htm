Layer (Hexagonal1)
Hexagonal1 ist ein Layer mit sechseckförmigen Tiles. Sechsecke sind oben/unten flach und haben seitliche Spitzen. Solche Layer werden ab und zu in einigen Spielen verwendet und können relativ einfach dargestellt werden. Die X/Y-Koordinaten sind etwas chaotisch angeordnet. Eigenschaften


Tileset:
Ein Tileset kann mit Tileset Editor angelegt und hier jederzeit ausgewählt werden.

Tilelayer:
Tilelayer kann 4 Bit (15 Tiles), 8 Bit (255 Tiles), 12 Bit (4095 Tiles) oder 16 Bit (65535 Tiles) Tiefe haben. 8 Bit und 16 Bit sind jedoch etwas einfacher zu handeln (z.B. bei Pathfinding). Bei einer Änderung wird ein Layer neukalkuliert.

Datenlayer:
Datenlayer kann 0 Bit, 1 Bit (1 Wert), 2 Bit (2 Werte), 4 Bit (15 Werte) oder 8 Bit (255 Werte) haben. Bei einer Änderung wird ein Layer neukalkuliert.

Layername:
Layername ist immer optional. In einem Spiel kann man so einzelne Layer unterscheiden.

Layerpos X/Y:
Layerposition wird in Pixel angegeben

Layerbreite, Layerhöhe:
Layerbreite und Layerhöhe wird in Tiles angegeben. Bei einer Änderung wird ein Layer neukalkuliert.

Parallax X/Y:
Parallax ist normalerweise immer 100%. 1%-99% lassen es langsamer scrollen. 101%-200% lassen es schneller scrollen. 0% fixiert den Layer.

Tile Shift:
Ist Tile Shift aktiviert, dann wird die Koordinate 0,0 versetzt dargestellt.

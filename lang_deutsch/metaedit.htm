Meta Editor
Meta Editor kann bis zu 65535 Byte an Zusatzinformationen einem Layer zuweisen. Diese Daten kann dein Programm später weiterverarbeiten (Zahlenwerte, Texte oder sogar Skripte).

BINÄR: Dieser Modus erinnert an einen HEX-Editor. Jedoch steht in jeder Zeile nur ein Byte. Short- oder Integer-Werte werden abhängig von dieser Offsetposition angezeigt. Jeder Wert kann manipuliert werden.

Klicke auf Plussymbole, um neue Bytes hinzuzufügen (Pfeil in der Symbolleiste zeigt ob diese über/unter der Markierung hinzugefügt werden). Klicke auf Minussymbole, um Bytes wieder zu entfernen.

ASCII: Dieser Modus unterscheidet sich kaum von einem Texteditor - schreibe einfach deine Texte hier rein.

Nach einem Klick auf BINÄR oder ASCII werden die Daten konvertiert. Es erscheint ein Hinweis, wenn dadurch Daten verloren gehen werden.

Die Daten können auch importiert oder exportiert werden. Alle Daten können auch nach einem Klick auf NEU entfernt werden. Nach einem Klick auf OK werden alle Änderungen übernommen.

Schwarze Punkte zeigen nur ein 2 oder 4-Byte-Raster an - ansonsten haben diese keine Funktion.

Meta Editor ist nicht in der Free-Version enthalten!

Objekte
Der Map-Editor bietet eine Unmenge an Objektarten. Erst dadurch kann der Editor universell eingesetzt werden. Die Symbolleiste bietet folgende Icons:

Objekt hinzufügen

Objekt entfernen

Objekt nach oben

Objekt nach unten

Eigenschaften

Nach einem Klick auf Objekt hinzufügen erscheint dieses Menü. Wähle dort ein Objekt aus. Ein Tileset kann ebenfalls sofort zugewiesen werden. Ein Doppelklick auf die Liste entspricht dem Klick auf OK-Button.



Neue Objekte erscheinen dann sofort im Objekt-Menü. Dort können Objekte ausgewählt werden. Nach einem Doppelklick erscheint Eigenschaften-Dialog. Bei einem Klick auf das Icon kann zwischen sichtbar und unsichtbar umgeschaltet werden. Aktives Objekt wird jedoch immer angezeigt!



Hinweis: rote Sternchen hinter Objektnamen deuten auf zugewiesene Matadaten!

Hintergrund
Hintergrund-Objekt ist ein fester Bestandteil jeder Karte und kann nicht gelöscht werden. Es speichert einige Informationen zu der untersten Schicht einer Karte. Es kann eine Füllfarbe oder eine Textur gewählt werden.



Hintergrund:
Hier kann Füllmodus eingesstellt werden (Ohne, Farbe, Bild)

Farbe:
Hier kann Füllfarbe eingestellt werden.

Grafikdatei:
Hier kann eine Textur ausgewählt werden. Button mit Pfeil löscht es aus dem Speicher.

Grafikpos X/Y:
Hier kann Grafik-Verschiebung eingestellt werden.

Parallax X/Y:
Parallax ist normalerweise immer 100%. 1%-99% lassen es langsamer scrollen. 101%-200% lassen es schneller scrollen. 0% fixiert den Layer.

Kartenbreite, Kartenhöhe:
Optionale Angabe zur Größe der Karte - kann im Spiel ausgelesen und verwendet werden.

Passwort:
Passwort für Kartenverschlüsselung. So werden Karten vor Manipulation geschützt.

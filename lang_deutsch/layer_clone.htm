Layer (Klon)
Klon-Layer enthalten keine eigene Daten. Die Daten werden einfach von einem anderen Layer verwendet. Alle Änderungen am Referenz-Layer wirken sich sofort auch auf Klon-Layer aus. Klon-Layer verbrauchen zudem nur wenig Speicher. Position kann aber individuell eingestellt werden.



Layer:
Wähle hier bereits erstellte Layer aus. Das wird als Referenz für Klon-Layer benutzt.

Objektname:
Objektname ist immer optional. In einem Spiel kann man so einzelne Objekte unterscheiden.

Objektpos X/Y:
Objektposition wird in Pixel angegeben

Parallax X/Y:
Parallax ist normalerweise immer 100%. 1%-99% lassen es langsamer scrollen. 101%-200% lassen es schneller scrollen. 0% fixiert den Layer.

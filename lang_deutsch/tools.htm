Werkzeuge
Die Werkzeuge sind universell und funktionieren mit Frames, Animationen oder Datenlayern zusammen. Fast alle Werkzeuge sind nur für Layer-Objekte bestimmt und zeigen darum keinerlei Wirkung auf geometrische Objekte und Bildobjekte. Beachte, dass rechte Maustaste einen Tilewert herauspicken kann - so ähnlich wie in einem Malprogramm.

Auswählen
Wählt ein Objekt direkt im Editor aus. Geometrische Objekte werden zuerst ausgewählt. Danach folgen Bildobjekte und dann Layerobjekte. Es ist auch möglich Objekte zu verschieben und Größe zu ändern. Halte linke STRG-Taste, um die Verschiebesperre zu unterbrechen.

Markieren
Markiert einen rechteckigen Bereich. Dies funktioniert nur mit Layer-Objekten. Benutze Copy&Paste-Icons, um den gewählten Ausschnitt zu manipulieren.

Freihand
Zeichnet ein ausgewähltes Tile in einen Layer ein.

Rechteck
Zeichnet einen rechteckigen Rahmen in einen Layer ein. Hierzu wird der Bereich mit dem ausgewählten Tile gefüllt.

Block
Zeichnet einen rechteckigen Block in einen Layer ein. Hierzu wird der Bereich mit dem ausgewählten Tile gefüllt.

Sequenz
Zeichnet einen rechteckigen Block in einen Layer ein. Zuerst wird ausgewähltes Tile eingezeichnet und dann alle nachfolgende Tiles. So kann man schnell Tiles setzen, die ein größeres Bild ergeben.

Füllen
Füllt einen Bereich mit dem ausgewählten Tile aus. Bei großen Karten kann dieser Vorgang einige Sekunden dauern. Es werden nur "gleichfarbige" Flächen gefüllt.

Zauberstift
Funktioniert wie Freihand. Es werden jedoch auch benachbarte Tiles manipuliert. Ist der Nachbarwert geringer, dann wird Tile-1 gesetzt. Ist der Nachbarwert höher, dann wird Tile+1 gesetzt. So können Landschaften einfacher erstellt werden. Dazu werden jedoch speziell angepasste Tilesets benötigt. Teste diese Funktion zuerst an einem Datenlayer.

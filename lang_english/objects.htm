Objects
Map editor can handle many different objects. This makes the editor universal. You will find these icons in the toolbar:

add object

remove object

object up

object down

properties

After you click on "add object", the following menu pops up. Choose an object from the list. You can also assign a tileset to the object. A doubleclick is identical to a click on the OK-button.



You will find new objects in the object menu and you can select an object from this list. A doubleclick on an object will display a properties-window for that object. You can also switch the object between visible/invisible if you click on the icon to the left of the object name. But you will always see the active object!



Note: red star behind layer name display, that metadata was assigned to this object.

View
You can hide some details for better viewing. You can find these icons in the toolbar:

Separate view on/off
If you work on complex maps with many layers - sometimes one layer can cover another layer. This feature hides all objects and only displays the active object.

Zoom on/off
Zoom in to show more detail.

Grid on/off
Map editor draws a grid on the active layer, but you can disable this feature.

Parallax on/off
If you use parallax scrolling, sometimes you must scroll long distances. It can be difficult to edit large maps, so you can turn this feature on and off. Note: To observe parallax scrolling, you will have to set a different value (something other than 100%) into at least one layers parallax setting.

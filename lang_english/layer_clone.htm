Layer (clone)
The clone layer does not contain any data of its own. This object uses data from another layer. However, you can set the clone layers position/parallax seperately. All changes to the reference layer modify the clone layer in realtime. The clone layer doesn't use much memory.



Layer:
Select a layer here. This layer will now be used as the reference layer.

Objectname:
Objectname is always optional. You can read this value in your program.

Objectpos X/Y:
Object position (in pixels).

Parallax X/Y:
Parallax is set to 100% by default. Use 1%-99% for slower scrolling. Use 101%-200% for faster scrolling. 0% fixes the background object.

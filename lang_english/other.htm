Other
New
Creates a new map and deletes all data. Only the clipboard retains information. You can't undo this action.

Load
Displays a filebox and allows you to open one map-file. If the map file is encrypted an inputbox will popup - write your password into it.

Save
Saves the complete map to a file. Hold the left-shift-key to overwrite the opened file without showing a filebox.

Setup
Displays the setup-window. You can change colors or the language.



Help
Shows this help file.

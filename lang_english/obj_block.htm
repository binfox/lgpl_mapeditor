Object (block)
Block object is a 1x1 layer - but it is faster. These objects are always glued to a layer-object and you just set the tile-position of where this block object should be displayed. If you move the reference layer - all assigned blocks to the layer will also move. You can also play an animation independent from the master animation (different timing).



Layer:
The reference layer that this block-object is glued to.

Tileset:
After you create a tileset with the tileset editor you can choose a tileset here.

Animation:
Set the animation mode here (standard, paused, forward, backward). This is only available if you've assigned an animation tile to the image.

Startframe:
Set the starting frame here. This is only available if you've assigned an animation tile to the image.

Imagename:
Imagename is always optional. You can read this value in your program.

Tilepos X/Y:
Sets the position on the reference layer.

Adjustment X/Y:
You can calibrate this object and move it a few pixels up/down/left/right. This is useful for iso/hex layers. Use minus numbers to move left or up.

Mask image:
You can draw masked images (transparent mask color). Enabled by default.

Object (line)
Line is a geometrical object that is invisible in your game. The function of this object is user-defined. You can use it for vector collisions (Sonic: The Hedgehog) or to mark the path of a conveyor/transporter.



Objecttype:
Object selection (point, line, rect, oval).

Objectname:
Objectname is always optional. You can read this value in your program.

Objectpos X/Y:
Object position (start position) (in pixels).

Endpos X/Y:
End position (in pixels).

Parallax X/Y:
Parallax is set to 100% by default. Use 1%-99% for slower scrolling. Use 101%-200% for faster scrolling. 0% fixes the background object.

Universal Map Editor
With 'Universal Map Editor', you can create new game maps in a short amount of time. 'Universal Map Editor' is designed for most tilebased games (arcade, puzzle, logic, maze, race, shooter, strategy, jump&run, platform ...). This program exists in two versions. This version is "only" the FREEversion, but a commercial version (PROversion) is available - for large maps. PROversion can also create isometric and hexagonal maps.

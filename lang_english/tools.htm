Tools
All tools are universal and work with frames, animations or datalayers. Most tools can only be used with layers and you will not see any changes to image objects or geometrical objects. Note: You can pick a tile with the right mouse button.

Select
Selects an object directly from the editor. Geometrical objects have highest prority - layer objects have lowest priority. You can also move objects or modify the size of geometrical objects by holding down the left-control-key while clicking/dragging.

Mark
Marks a rectangular area. This only works with layer objects. Use cut/copy/paste functions once you have selected something.

Draw
Draws the selected tile into a layer. You can click or drag the mouse to draw.

Rect
Draws a rectangular border (an outline) into a layer.

Block
Draws a rectangular block into a layer. The selected area is filled with the selected tile.

Sequence
Draws a rectangular block into a layer. The selected area is filled with more than one tile. At every step the map editor increases the frame number.

Fill
Fills the area clicked on by overwriting similar tiles with the selected tile. This action may take a few seconds on large maps.

Magic pen
Similar to the Draw tool but manipulates neighboring tiles. If the neighboring tiles value is lower, the map editor sets the neighboring tile to the ((selected tile)-1) tile. If the neighboring tiles value is higher, the map editor sets the neighboring tile to the ((selected tile)+1) tile. You need a special tileset for this tool for best results. This tool also works on datalayers.

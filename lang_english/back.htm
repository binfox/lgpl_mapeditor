Background
The background-object is a fixed part of every map file. You can't remove this object. This object also stores some information for the background layer. For example, you can set the background color or a background image.



Background:
Set the fill mode for the background (without, color, image, masked image)

Color:
Set a background color (or a mask color) here.

Imagefile:
Set a background image here. The button with the left-arrow will remove the image from the background.

Imagepos X/Y:
Set the offset position of the background image here.

Parallax X/Y:
Parallax is set to 100% by default. Use 1%-99% for slower scrolling. Use 101%-200% for faster scrolling. 0% fixes the background object.

Mapwidth, Mapheight:
Optional values for limiting map size - you can read these values in your game.

Password:
Password for map file encryption. This won't allow anybody to be able to use your maps in other games.

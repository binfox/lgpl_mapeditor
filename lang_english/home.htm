Universal Map Editor is a special program made for game developers and level creators. It was designed for most 2D tilebased games. Arcade, puzzle, logic, maze, race, shooter, strategy, jump&run or platform games are possible. This editor can reduce the development time by 50%.


Create rectangular, isometric or hexagonal maps with up to 100,000,000 tiles per layer - of course with animations and parallax scrolling - using simple tools and intelligent features. Experts enjoy datalayer, metadata, animated sprites and geometrical objects.


All file-specifications are given to you. Sourcecode (written in BlitzBasic) is also available. This program exists as a FREEversion and a PROversion.

System requirements: Pentium 300 Mhz, 64MB RAM, 8MB VRAM, Soundcard, DirectX, Windows 95,98,ME,NT4,2000,XP.

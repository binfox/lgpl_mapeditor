Meta editor
The Meta editor can save up to 65535 bytes of meta information per object/layer. Your program can read this information (value, string or script).

BINARY: this mode is smilar to a HEX-editor. The major difference is that you only see 1 byte per line. Short and integer values are displayed for each line. You can manipulate each value.

Click on the plus-icons to add some new bytes (Arrow in the toolbar determines if new bytes are added above or below the selection). Click on the minus-icons to delete some bytes.

ASCII: this mode is similar to a normal text editor - write your text in the text area.

If you click on the BINARY or ASCII tabs, map editor will convert the data for the destination mode. A warning-message will appear if you will lose information during the conversion.

You can import or export your data. Click on OK to return to the main window.

The black dots just display 2 or 4 byte patterns - they have no other function.

Meta editor is not included in the FREEversion!
